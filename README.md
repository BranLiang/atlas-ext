# IssueExplorer

Tailored for readers of Ruanyifeng's 科技爱好者周刊, it enhances each issue by providing relevant contents.

### Quick Start

1. **Install Bun:**

   ```bash
   curl -fsSL https://bun.sh/install | bash
   ```

2. **Install Dependencies:**

   ```bash
   bun install
   ```

3. **Build for Firefox:**
   ```bash
   bun run package
   ```
